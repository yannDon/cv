  • HTML: J'ai appris les concepts de base de l'HTML, tels que les balises, les attributs et la structure d'une page web. J'ai compris comment utiliser des éléments tels que les titres, les paragraphes, les images et les liens pour structurer le contenu d'une page web.

  • CSS: J'ai appris les concepts de base du CSS, tels que les sélecteurs, les propriétés et les valeurs. J'ai compris comment utiliser le CSS pour mettre en forme le contenu structuré par l'HTML, en utilisant des styles tels que la mise en forme de texte, la disposition des éléments et la gestion des couleurs.

  • JavaScript: J'ai appris les concepts de base du JavaScript, tels que les variables, les fonctions et les boucles. J'ai compris comment utiliser le JavaScript pour ajouter des fonctionnalités interactives à une page web, comme la validation de formulaire, la manipulation du DOM et les interactions utilisateur.
